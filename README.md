## Specifications
### BOM
- 1x Seeed Xiao RP4020
- 1x EC11 Rotary Encoder
- 4x Cherry MX switches
- 4x Blank Keycaps
- 4x SK6812 MINI-E LEDs
- 4x Screws

### Others
- Custom PCB
- QMK Firmware
- Case.STL
